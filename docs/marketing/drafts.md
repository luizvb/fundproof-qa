## Product description

FundProof QA gives IR, content operations and compliance teams a source-addressed review queue for fund-assistant QA sets. Every support and conflict finding points back to a document, page and exact span. Missing evidence stays inconclusive.

## Tool description

Run the same fixed checks in CI or on a laptop. `fundproof-lint` reads JSON locally, writes a machine-readable report and makes no network calls.

## Workspace boundary

Keep libraries versioned, compare release diffs, schedule recurring checks and apply client-configured policies with collaboration and audit history.
