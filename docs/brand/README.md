# Brand direction

Reading this as: a greenfield landing with a functional product for asset-manager IR, compliance and AI-governance teams, using institutional, evidence-first language and an editorial compliance surface.

- `DESIGN_VARIANCE: 4/10`
- `MOTION_INTENSITY: 3/10`
- `VISUAL_DENSITY: 5/10`

## Identity

Name: FundProof QA

Tagline: Prove what your fund assistant can answer.

Voice: precise, restrained, review-oriented.

Mark: two offset document leaves connected by a cobalt evidence link. It is rendered with simple CSS geometry so it remains crisp at small sizes.

## Tokens

- cold paper: `#eef1f4`
- raised paper: `#f8fafb`
- graphite: `#18212b`
- muted graphite: `#52606d`
- desaturated cobalt accent: `#345f9d`
- cards: 14px radius
- interactive controls: pill radius
- motion: feedback transitions and loading shimmer only

Light and dark values use CSS variables and follow system preference. Reduced-motion disables the shimmer and shortens transitions.
